智慧型代理人程式碼相似度偵測系統
Intelligence Agent for Programs Similarity Detection System
智慧代理人、程式碼抄襲、相似度
程式碼抄襲問題影響深遠，包括學習品質降低、商業侵權等，但在檢查是否抄襲的過程既費時且費力。因此，本研究利用智慧代理人取代人工方式，以全面性的比對方法做客觀判定，在追求提昇相似度同時，更注重是否符合事實。   本研究利用智慧代理人應用於比對程式碼抄襲作業上，作為取代浪費人力及時間的人工檢查方式。在我們實作的Cpo(Code Police Officer)系統賦予代理主要的智能有偵測錯誤檔案、比對程式碼相似度及回報結果等。最後我們也將Cpo與Sid(A Software Integrity Diagnosis System)系統做插入無意義程式碼實驗比較，Cpo在文字、結構及變數相似度比較皆獲得較好結果。    本研究專注於程式碼相似度演算法、計算公式及系統實作及相關實驗比較上面，目前雖然已建構智慧代理人基本雛型架構，仍尚有許多特點未發揮，如偵測一份標準程式碼可以得到文字、結構及變數權值等，都是未來需要再加強延伸的功能。
The problem of program plagiarism influences deep and far. It includes the low quality of learning and business tort, etc. But it takes a lot time and requires strenuous effort to check plagiarized program. For this reason, we use intelligent agent to substitute for manual way. Agent takes objective and wide method to detect program. It is not only to purse high similarity, but also to fit in with fact.    In our research, we have utilized intelligent agent to detect the similarity of program to substitute for human checking plagiarized program. And we have already implemented a detection system of program similarity named Cpo(Code Police Officer) to detect plagiarized program. The last, compare Cpo and Sid(A Software Integrity Diagnosis System) by inserting unrelated program, Cpo has got the better performance.    We focus on the algorithm of program similarity and method of calculation and implementation of system now. So just construct the basic system architecture of intelligent agent. There are many advantages that don’t be implemented now. For example, to detect a standard file that can get the weight value of text、structure and variable.
