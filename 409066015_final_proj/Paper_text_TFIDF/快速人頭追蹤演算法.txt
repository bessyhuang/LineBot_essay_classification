快速人頭追蹤演算法
A fast tracking algorithm of human head
人頭追蹤、橢圓形模板匹配、差分影像
摘  要本論文提出一種橢圓形模板匹配方式的快速人頭追蹤演算方法及快速人頭輪廓定位追蹤之數位監視系統，本系統經由WebCam影像擷取的方式，如果偵測到畫面中有人物進入監視範圍時，可以快速的將人頭輪廓位置定位出來，以提高數位監視系統效率。其中系統可以分成三部分，第一部分是利用影像前置處理，過濾WebCam擷取畫面的影像雜訊，第二部分利用差分影像演算法將移動中量測物體標示出來，第三部分將輸入影像經由邊緣偵測(Edge Detection) 將量測物體的影像邊緣找出來，並使用簡易橢圓演算法達到人頭輪廓之初步定位，由於橢圓形模板匹配的偵測方式需要龐大的計算量，所以本論文提出標示演算法與減少模板數量的方式，以減少追蹤人頭輪廓時資料量的計算。
ABSTRACTThis paper purposed a fast head tracking approach using ellipse matching method. Our system contains three sub-systems. 1. Image preprocessing  2. Object extracting 3. Ellipse matching. The purpose of image preprocessing sub-system is to remove noises. The Object extracting sub-system extracts the possible object using frame difference techniques and locates the preliminary positioning of the object to reduce the search space of ellipse matching. In the ellipse matching we used sobel operators for edge detection of source image, after the edge detection procedure we locate the head outline using an ellipse matching to confirm if the object of interest is indeed the human head. We purpose a moving object marker algorithm for ellipse matching to reduce the time required of head tracking.
