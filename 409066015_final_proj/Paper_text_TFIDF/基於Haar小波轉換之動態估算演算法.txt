基於Haar小波轉換之動態估算演算法
A Motion Estimation Algorithm base on Haar Wavelet transform
動態估算、動態補償、區塊比對
本專題研究是在連續影像中將影像裡順序兩張影像，利用影像在不同頻率中的各數值比對，作為找尋最相似的區塊比對演算法(BMA)的先行搜尋，來達到搜尋最匹配的區塊。在一影像中，邊緣的影像會落在高頻的區域，因此之前學者提出了EAS (Edge- Assisted Algorithm )演算法，其利用了 SOBEL 中微分與低通濾波來強化邊緣的影像，再利用其來搜尋相近的邊緣圖形，來當做BMA(Block Match Algorithm )的搜尋起點(start point )。這樣的方法很成功改進了方塊搜尋法和T度下降搜尋法，使其在搜尋最匹配的區塊時，避免了只找到局部區域中最匹配的區塊。由於許多的BMA演算法，是基於向量的分佈，去設計適合影像中向量分佈機率的搜尋樣式，依照搜尋樣式循序的找尋最適合的區塊，但是實際的狀況並非如此，因此通常並不能搜尋到最匹配的區塊，反而陷入局部中較匹配的區塊。此專題研究提出的方法，可以搭配任一種BMA，並簡化了很多程序，其主要是利用 wavelet 轉換，達到圖形中影像高頻與低頻的分離，利用這些數值在搜尋視窗中(search window)來搜尋到最接近的區塊，當作傳統BMA 的搜尋起點(SP)，來達到較好的搜尋結果。
This Paper is studied Motion Estimation to search match block between Reference Frame and Current Frame in sequence image which use image Value in different frequency band before BMA(block match algorithm).In other approach propose EAS(Edge- Assisted Algorithm) Algorithm which used Sobel filter to enhance Edge area ,then compare the block between two frame .After fined similarity block , that coordinates will be start point in BMA . this method modify diamond search and block base gradient search successful to avoid fined best match block in local area .Because many BMA is designed search pattern, in which used probability of motion vector base on distribution of the Motion vector, that will search best match block follow search pattern. But it is not true in real world, often can not search best point in search window, and fall into local area to fined minimum point. In this paper, the method can collocate with any BMA and reduce complex process, which used wavelet transform to divide high band and low band, then use these value to compare best close block in search window, these position will be search Start point in BMA to reach search best point in sequence image.
