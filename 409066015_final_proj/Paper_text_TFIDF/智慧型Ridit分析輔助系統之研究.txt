智慧型Ridit分析輔助系統之研究
A Study on the Intelligent Support System for Ridit Analysis
Ridit分析、Java統計分析系統、電腦輔助教學
Ridit分析是由Irwin Bross於1958年所提出的分析方法，該方法可用於分析具有順序特徵且不具特殊理論分配假設的類別資料。自從Bross提出Ridit分析以來，Ridit分析已經廣泛運用在不同的領域中。但是Ridit分析圖表的繪製不易，因此本研究以Ridit分析為核心，用Java程式語言為開發平台撰寫智慧型分析輔助系統。本系統可對原始資料表進行存取、修改與計算次數分配。也可直接以次數分配資料表做存取、修改與進行Ridit分析。分析後提供Ridit分析結果報表、信賴區間圖表與智慧型輔助分析功能，以利解讀分析結果。除此之外，本系統亦提供參考手冊功能，對Ridit分析的由來、方法與應用作陳述，協助使用者更容易了解與應用Ridit分析。整體而言，本系統能以視覺化的方式提供Ridit分析結果及繪製圖表，配合參考手冊與智慧型分析輔助功能，協助使用者更容易了解Ridit分析與應用分析結果。
Ridit Analysis was introduced by Irwin Bross in 1958. This method can be used to analyze categorical data which have ordinal characteristic and does not assume a theoretical distribution. Ridit Analysis has been put in use in many fields since its introduction. This research is motivated partially by the fact that the graphs for ridit analysis are not easy to make. In this research we use Java language to develop an intelligent support system that facilitates Ridit Analysis.This system can take raw data tables and frequency data tables as input and can produce computation results and graphs. Furthermore, the system also provides intelligent support functions that can help users to interpret the results. Besides, this system provides a reference guide that gives an introduction of how to use Ridit Analysis.On the whole, this system can provide visual analysis results and graphs. With the reference guide and intelligent support functions, this system also helps users understand and use Ridit Analysis more easily.
